# DJ Morocco Landing Page

A vibrant, modern landing page for a Moroccan DJ brand that reflects the energy and spirit of Moroccan nightlife and electronic music.

## Features

- Fullscreen hero section with animated background
- Moroccan electronic fusion music style description
- Events section with upcoming shows and past performances
- Interactive music player with sample tracks
- Instagram feed integration
- Contact & booking form
- Dark theme with neon/glow effects and vibrant gradients
- Mobile-responsive design
- Language toggle (English/French/Arabic)

## Technologies Used

- HTML5
- CSS3 with animations and responsive design
- JavaScript for interactive elements
- Font Awesome for icons
- Google Fonts

## Getting Started

Simply clone the repository and open the `index.html` file in your web browser to view the landing page.
